Car Price Prediction using Machine Learning

Overview
This project is a Machine Learning model that predicts the selling price of used cars based on various features such as year of manufacture, present price, kilometers driven, fuel type, transmission type, and ownership history.
The goal is to help users estimate a fair market price for used cars using regression algorithm

Objective
- To analyze used car dataset  
- To identify key factors affecting car price  
- To build a regression model for accurate price prediction  
- To improve decision making for buyers and sellers 

 Dataset Description
The dataset contains the following features:
- Car name  
- Year of manufacture  
- Present price  
- Selling price  
- Kilometers driven  
- Fuel type (Petrol/Diesel/CNG)  
- Seller type (Dealer/Individual)  
- Transmission type (Manual/Automatic)  
- Number of previous owners
